Лаборатори Ажил 2 Бие даалтын Веб хуудас

Нийтэлсэн Э.Мөнхжин2020 оны 2р сарын 23

Энэхүү лабораторийн ажлаар бүгдээрээ 5 хуудастай өөрсдийн танилцуулга вэб сайтыг хийнэ ээ. Энэхүү 5 хуудсанд Нүүр, Миний тухай, Миний хобби, Гэр бүл, Холбоо барих, зэрэг хуудсууд багтна. Энэхүү хуудсуудыг хийхдээ 2-р лекцээр үзсэн HTML tag кодчилолыг ашиглана. Ашиглах боломжтой tag-ууд: h1, h2, h3, h4, h5, p, ul, ol, li, a, img, table, form болон бусад ..

Вэб хуудсыг хийхдээ: Өөрийн үнэн зөв мэдээллээр байжуулж хийх 2. Өнгө дизайны зохицол сайтай байх 3. Аль болох олон таг өөрсдөө бие даан судлаж олон ашиглах Хийгээд дууссан бол github дээрээ тавиад онлайнаар шалгуулах боломжтой линкээ үүсгээд энэ assignment-рүү илгээнэ үү.
